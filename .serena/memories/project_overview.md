# Project Overview

## Purpose
Canvas Image Viewer is a lightweight, client-side web application for viewing images on HTML5 Canvas and downloading them to your device.

## Key Features
- Drag & drop upload for JPG, PNG, and WebP images
- HTML5 Canvas rendering
- Download canvas images directly to device
- Dark mode support
- Client-side only - no server uploads required

## Tech Stack
- **Framework**: Next.js 16.0.1 (App Router)
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS v4
- **UI Components**: react-dropzone for file uploads
- **Runtime**: Node.js 20+
- **Package Manager**: Bun (as indicated in README)

## Project Type
This is a Next.js 16 application using the App Router architecture with TypeScript and Tailwind CSS v4.
